How it works

This demo uses a tiny content-based model. Each track has features like danceability, energy, valence, and tempo (scaled 0–1). When you pick a seed song or set preferences, we calculate cosine similarity between that feature vector and every track, then return the top matches.

It’s intentionally simple so you can see what’s happening end-to-end without hidden training or external services.
